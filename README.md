# 📘 Descripción del Repositorio

Este repositorio recopila writeups técnicos y reportes ejecutivos sobre laboratorios de Ciberseguridad Defensiva realizados como parte del camino hacia el rol de SOC Analyst (Nivel 1).
Incluye simulaciones de incidentes reales, análisis forense, detección de TTPs mediante MITRE ATT&CK y reportes híbridos orientados a equipos técnicos y gerenciales.

## 🛡️ Objetivo

- Consolidar experiencia práctica en detección, análisis y respuesta a incidentes.

- Desarrollar capacidad de comunicación ejecutiva en entornos SOC.

- Documentar evidencia técnica y lecciones aprendidas de cada escenario.


### 🧩 Cada Reporte Incluye
- Resumen Ejecutivo: Impacto y contexto inicial

- Evidencia Técnica: Log sources, TTPs y hallazgos clave

- Acciones Tomadas: Cronología SOC

- Recomendaciones: Hardening y detección futura (SPL)

## ⚙️ Tecnologías y Fuentes
- SIEM: Splunk, Wazuh

- Logs: Sysmon, Event Viewer, Zeek, Syslog.

- Frameworks: MITRE ATT&CK, NIST 800‑61

- Herramientas: TryHackMe, LetsDefend, CyberDefenders
